If you need help, first watch the video above, it covers just about everything. If SeekCut doesn’t seem to be working, make sure your video is encoded in either the H.264 or ProRes codecs. If you apple-I the video file and you see “AVdn” as a codec, it’s a DNx file and SeekCut cannot work with it. Convert it into a H.264.
